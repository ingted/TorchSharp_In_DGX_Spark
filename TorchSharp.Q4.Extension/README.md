# TorchSharp.Q4.Extension

F#-first Q4 extension for TorchSharp with:

- `NF4` and `NVFP4` schema support
- backend dispatch (`nf4-kernel`, `nvfp4-kernel`, `dequant-matmul`)
- optional native bridge to `/workspace/nvfp4_native/libNVFP4.so`
- runtime diagnostics and policy controls

Primary package id: `FAkka.TorchSharp.Q4.Extension`
